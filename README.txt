	To start NumeriX, just unzip all the files into a folder and run the "run.bat" file or double click the "Mint.jar" File in the libs directory.

MINT is built using the following tools.
	* Sun Java
	* Antlr - Compiler Compiler
	* jmat - Matrix Package
	* xfunctions - Graph plotting utility
	* gnuprolog - prolog-java interface
