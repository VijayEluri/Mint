Syntax and Structure of Mint

Statements:

  • A program can contain zero or more statements.
  • Every statement should end with an SEMI COLON( ; )

Eg: a = 1 + 2 * 3 ;

Variables

  • Variables should start with alphabets and can contain any characters
  • Variables can be of any length
  • Variables should not contain keywords such as function names

Eg: a, a123, a_123 -> valid

123a, ..aa -> invalid

Numbers

  • Numbers can be of the following format
    • Integers
    • Double
      • Eg: 123.123, 123E12, 123E-12
    • Complex Numbers
      • Eg: 1+2i, 2i etc
    • Big Integers ( Order of 10^N )

Arrays

  • Arrays in MINT are 1D and 2D arrays.
  • Arrays are created using SQUARED BRACKETS( [...] )
  • Arrays can contain data of same type or of mixed type
  • Columns are sepereated using COMMA ( , )
  • ROWS are sepereated using SEMI COLON ( ; )

1D array:

a = [ 1 , 2, 3 ] -> array with same data type

a = [1, 'cat', 1.123] -> array with different data type

2D Array:

a = [ 1,2,3; 1,2,3; 1,2,3] -> a 3x3 array with same data type.

Complex Number

Mint also provides easy way to represent complex numbers.

  • Imaginary number 'i' is used to define a complex number.
  • 'i' by default a constant in MINT
  • Basic arithmetic operations are applicable

Eg:

a = 2*i -> complex number value (0 + 2i )

a = 1+2*i -> complex number value(1+2i).

 

Big Numbers

Big Numbers are numbers that are in the order of 10^N, where N > 10.

  • Function named big(..) is used to define the number as big number

Eg:

a = big(10);

b = a ^ 1000;
