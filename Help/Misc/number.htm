Number

A number in Mint may be one of the following

  • Double Number
  • Integer Number
  • Complex Number
  • Matrix
  • BigNumber
  • EigenValue Matrix
  • LUDecompostion Matrix

Real Number:

  • 1
  • 1.5
  • 1.32243234
  • 123123.80909

Complex Number:    A number with the imaginary part. i or j is used to represent a complex number

  • 1 + 2i
  • 2-3j
  • 2.234 + 567.456j
  • 234j

Matrix :

[1,2,3] - > 1 x 3 matrix

[1,2,3;3,2,1] -> 2 x 3 matrix

[1,2,3;4,5,6;7,8,9] -> 3x3 matrix

COMMA ( , ) -> used to seperate columns

SEMI COLON( ; ) -> used to seperate rows

BigNumber :

Represents a very big Number of order 10^N, i.e 100000000000000000......0000 etc.
