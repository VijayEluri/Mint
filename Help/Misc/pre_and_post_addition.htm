pre and post addition ( ++ )

Increments a number by one, either prefix of post fix

Number++ or ++Number

Syntax:

2.3++ or ++2.3

Return Type: Double Number

(2 + 5i) ++ or ++(2 + 5i)

Return Type: Complex Number

BigNumber++ or ++BigNumber

Return Type: BigNumber

Matrix++ or ++Matrix

Return Type: Matrix
