multiplication ( * )

It multipies two double numbers , matrices , complex numbers , bignumbers

Syntax:

2.3 * 4.5

Return Type: Double Number

(2 + 5i) * (4 - 9i)

(2 + 5i) * 4

(2 + 5i) * 9i;

Return Type: Complex Number

BigNumber * BigNumber

BigNumber * Double Number

Return Type: BigNumber

Matrix * Matrix

Matrix * Double Number

Return Type: Matrix

String * Number => Copies the given string no of times the Number

Return Type: String
